\section{Características}
\begin{itemize}
\item Tipo de trabajo: Articulo de investigación sobre aplicación de métodos de minería de datos 
\begin{itemize}
\item Debe trabajar sobre una base de datos nacional o internacional (que incluya a Bolivia)
\item La temática de la base de datos es a elección del estudiante
\item Se debe incluir al menos 3 métodos vistos en la materia
\item Bases de datos con al menos 10 mil observaciones y 10 variables
\end{itemize}
\item Los trabajos no deben exceder las 15 paginas (sin contar anexos y caratula)
\item Al final se debe preparar una presentación del trabajo, (máximo 10 minutos)
\item (Opcional) Se valorara de forma positiva que el trabajo este en Latex/Sweave
\item Entregables
\begin{itemize}
\item Código de R
\item Articulo de investigación
\item Presentación
\end{itemize}
\end{itemize}

\section{Contenido mínimo del articulo de trabajo}
El contenido mínimo es:
\begin{itemize}
\item \textbf{Introducción}
\item \textbf{Objetivos}
\item \textbf{Motivación}
\item \textbf{Marco Teórico / Revisión de literatura} 
\item \textbf{Descripción de la base de datos}
\item \textbf{Metodología}
\item \textbf{Resultados y análisis}
\item \textbf{Conclusiones y recomendaciones}
\item \textbf{Referencias}
\end{itemize}

\section{Fechas Importantes}

\begin{itemize}
\item \textbf{Primera revisión (5 pts.):} 30 de marzo, hasta la descripción de la base de datos
\item \textbf{Segunda revisión (5 pts.):} 29 de abril, hasta la metodología
\item \textbf{Informe final y presentación (30 pts.):} 8, 10 de Junio
\end{itemize}
